# Bus Stop Finder (Valencia)

This is a small experimental project built to quickly find and check bus arrival times for the EMT Valencia bus service. It was created as a learning exercise and a personal utility.

### About This Project (For Developers)

If you're curious, this project was built with:

*   Next.js & React
*   TypeScript
*   Tailwind CSS
*   Data sourced from the public EMT Valencia API.

It serves as a simple example of how to interact with the EMT API and display real-time data. However, it is not intended for production use.
